Project Smart Home Monitoring Berbasis ESP32 Mini

Membuat sistem monitoring rumah jarak jauh menggunakan ESP32 Mini, sensor gerak, dan notifikasi real-time melalui internet.

Tentang Project

HOME ALONE Monitoring Project adalah sistem keamanan rumah sederhana berbasis IoT menggunakan ESP32 Mini. Sistem ini dapat mendeteksi pergerakan dan mengirim notifikasi ke smartphone melalui WiFi.

Komponen yang Dibutuhkan

  • ESP32 Mini
  • Sensor PIR (Motion Sensor)
  • Buzzer
  • LED Indicator
  • Resistor 220Ω
  • Kabel Jumper
  • Breadboard

Desain Project

Hubungkan:

Tutorial Pembuatan

1️⃣ Persiapan Software

  • Install Arduino IDE
  • Tambahkan Board ESP32
  • Install Library WiFi

2️⃣ Upload Program ke ESP32

#include <WiFi.h>

#define PIR 14
#define LED 27
#define BUZZER 26

void setup(){
  pinMode(PIR, INPUT);
  pinMode(LED, OUTPUT);
  pinMode(BUZZER, OUTPUT);
  Serial.begin(115200);
}

void loop(){
  int motion = digitalRead(PIR);
  if(motion == HIGH){
    digitalWrite(LED, HIGH);
    digitalWrite(BUZZER, HIGH);
    delay(3000);
  }else{
    digitalWrite(LED, LOW);
    digitalWrite(BUZZER, LOW);
  }
}

3️⃣ Testing Sistem

Nyalakan sistem dan gerakkan tangan di depan sensor PIR. Jika terdeteksi, LED dan buzzer akan aktif.
