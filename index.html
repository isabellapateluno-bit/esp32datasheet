2. Hardware Components & Theory

2.1 ESP32 Microcontroller

Dual-core 240MHz processor with built-in WiFi and Bluetooth. Features:

  • 520KB SRAM + 4MB Flash
  • 34 programmable GPIO pins
  • 12-bit ADC, DAC, touch sensors
  • Ultra-low power consumption (deep sleep: 10μA)

2.2 PIR Motion Sensor

Detects infrared radiation from human body (~37°C). Specifications:

  • Detection range: 3-7 meters
  • Field of view: 110° x 70°
  • Output: Digital HIGH/LOW (3.3V)
  • Dual-element sensor for motion direction detection

2.3 LED and Resistor Design

Using Ohm's Law: R = (Vs - Vf) / I For ESP32: Vs = 3.3V Vf ≈ 2V I ≈ 6mA R ≈ 220Ω

2.4 Buzzer Alarm

Piezoelectric buzzer with 5V active/passive options. Frequency range: 2-4kHz.

void setup(){
 Serial.begin(115200);

 WiFi.begin(ssid, pass);
 while (WiFi.status() != WL_CONNECTED) {
   delay(500);
   Serial.print(".");

 Blynk.config(auth);

void loop(){
 Blynk.run();
 
 if(digitalRead(PIR) == HIGH){
   digitalWrite(LED, HIGH);
   digitalWrite(BUZZER, HIGH);
   Blynk.logEvent("motion_alert", "Motion Detected!");
   delay(3000);
 } else {
   digitalWrite(LED, LOW);
   digitalWrite(BUZZER, LOW);
 }
}

10. Frequently Asked Questions

Q: How many sensors can I connect?

A: ESP32 supports up to 8 PIR sensors using GPIO multiplexing. For more sensors, use I2C expander.

Q: Can I use battery power?

A: Yes, with deep sleep mode and proper power management, 2x18650 batteries can last 30+ days.

Q: How to reduce false alarms?

A: Implement ML algorithm, adjust sensitivity, use multiple sensors, and add verification delay.

Q: Is cloud storage free?

A: Firebase offers free tier (1GB), Blynk has free plan with limitations. Local storage on SD card is always free.
