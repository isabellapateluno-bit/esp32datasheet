Project Overview

The HOME ALONE Monitoring System is an IoT-based security solution designed to monitor house activity remotely. Using an ESP32 Mini microcontroller with built-in WiFi, the system detects motion and triggers an alarm while sending alerts via internet.

Main Features:

  • Motion detection using PIR sensor
  • Real-time WiFi connectivity
  • LED & buzzer alarm system
  • Expandable for cloud notification

Required Components

1. ESP32 Mini

Main microcontroller with built-in WiFi and Bluetooth capability.

2. PIR Motion Sensor

Detects infrared radiation changes caused by human movement.

3. Buzzer

Produces audible alarm when motion is detected.

4. LED Indicator

Visual indication of intrusion detection.

System Architecture

The ESP32 connects to WiFi network and monitors PIR sensor input. When motion is detected, the system activates alarm outputs and optionally sends notification to cloud service.

Circuit Diagram

  • PIR OUT → GPIO 14
  • LED → GPIO 27 (with 220Ω resistor)
  • Buzzer → GPIO 26
  • VCC → 5V
  • GND → GND

Step-by-Step Tutorial

Step 1: Install Arduino IDE

Download Arduino IDE and install ESP32 Board Package.

Step 2: Upload Code

#include 

#define PIR 14
#define LED 27
#define BUZZER 26

void setup() {
  Serial.begin(115200);
  pinMode(PIR, INPUT);
  pinMode(LED, OUTPUT);
  pinMode(BUZZER, OUTPUT);
}

void loop() {
  int motion = digitalRead(PIR);

  if (motion == HIGH) {
    Serial.println("Motion Detected!");
    digitalWrite(LED, HIGH);
    digitalWrite(BUZZER, HIGH);
    delay(3000);
  } else {
    digitalWrite(LED, LOW);
    digitalWrite(BUZZER, LOW);
  }
}

Step 3: Testing

Power the ESP32 and move in front of the PIR sensor. The LED and buzzer should activate when motion is detected.
